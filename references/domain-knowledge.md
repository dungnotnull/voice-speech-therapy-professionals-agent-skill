# Domain Knowledge References

This directory contains authoritative domain knowledge used for grounding and RAG.

## Contents

### Assessment Frameworks

- **CAPE-V**: Consensus Auditory Perceptual Evaluation of Voice
  - 0-100mm Visual Analog Scale × 6 parameters
  - Severity categories: Normal (0-15), Mild (16-35), Moderate (36-55), Severe (56-100)

- **GRBAS Scale**: Grade, Roughness, Breathiness, Asthenia, Strain
  - 0-3 ordinal scale per parameter
  - Overall voice quality assessment

### Acoustic Norms

- **Jitter**: < 1.04% (frequency perturbation)
- **Shimmer**: < 3.81% (amplitude perturbation)
- **HNR**: > 20 dB (harmonics-to-noise ratio)
- **MPT**: ≥ 20s (males), ≥ 15s (females) (maximum phonation time)
- **s/z ratio**: < 1.4 (sustained /z/ to /s/ duration)

### Exercise Categories

1. **SOVT (Semi-Occluded Vocal Tract)**
   - Lip trills, tongue trills, humming, straw phonation
   - Benefits: Vocal fold efficiency, reduced impact stress

2. **Estill Voice Training™ Figures**
   - Figure 1: Chest Register
   - Figure 2: Falsetto Register
   - Figure 3: Speech Quality
   - Figure 4: Falsetto Quality
   - Figure 5: Twang
   - Figure 6: Oral Nasal Port

3. **Breath Support**
   - Diaphragmatic breathing
   - Costal expansion
   - Appoggio breathing

4. **Resonance**
   - Forward focus (placement)
   - Nasal resonance balance
   - Chant tone

5. **Articulation**
   - Consonant clarity drills
   - Vowel accuracy
   - Diction exercises

6. **Cool-Down**
   - Descending glissandi
   - Gentle vocal fry
   - Release stretches

### Pathology Types

- **MTD**: Muscle Tension Dysphonia
- **Vocal Fold Nodules**: Bilocal vocal fold lesions
- **Vocal Fold Polyps**: Unilateral fluid-filled lesions
- **Vocal Fold Cysts**: Unilateral encapsulated lesions
- **LPR**: Laryngopharyngeal Reflux
- **Presbyphonia**: Age-related voice changes
- **Spasmodic Dysphonia**: Neurological movement disorder

### Red Flags

- Sudden hoarseness > 2 weeks
- Dysphagia (difficulty swallowing)
- Odynophagia (painful swallowing)
- Hemoptysis (coughing blood)
- Cervical lymphadenopathy
- Unexplained weight loss
- Persistent pain

### Professional Subtypes

1. **Teacher**: High vocal load, classroom acoustics
2. **Podcaster**: Close-speaking, long-form recording
3. **MC/Host**: Live events, projection, ambient noise
4. **Actor**: Character voices, projection, performance schedule
5. **Singer**: Pitch range, belting, vocal stamina
6. **Call Center**: Continuous talking, headset use, targets
